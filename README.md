# 每日一书 · 学术精读

一个为研究生及以上读者准备的"近 2 周学术新作精读"站。

每天 8:30（北京时间）自动更新，覆盖 **社会学、人类学、历史学、政治学** 四个领域，
每领域挑一本新书或一篇论文，做成 Seminar 级精读卡。

## 卡片结构

每张精读卡包含：

1. **元信息**：封面、作者、出版社、ISBN/DOI、出版日期
2. **中英双语标题**
3. **一句话学科定位**
4. **导读**（150–200 字）
5. **核心概念释义**（3–6 个非懂不可的关键术语）
6. **重要插图**（≥2 张：原书图表 / 田野地风貌 / 关键事件地标）
7. **精读**（约 2000 字，章节级拆解 + 必读 / 可跳 / 关键脚注标注）
8. **学术对话**（与近 3 年代表作的关系：补充 / 修正 / 颠覆）
9. **优秀评论**（2–3 篇高质量书评的摘要 + 链接）
10. **推荐指数**（0–100，论据 40 / 新颖 30 / 可读 15 / 学术影响 15）

## 推荐指数怎么算

- **论据扎实度（40 分）**：研究方法、证据链、数据来源、田野厚度
- **议题新颖度（30 分）**：在同类研究中的位置、是否开辟新方向
- **可读性（15 分）**：行文、章节组织、可跳读性
- **学术影响（15 分）**：可能引起的对话与引用

## 时间窗口

- 主体窗口：最近 2 周（14 天）内初版
- 兜底窗口：4 周内高分新作（卡片顶部会明示"已回退"）

## 本地运行

```bash
python3 scripts/build_preview.py    # 生成预览版 candidates.json
python3 scripts/generate_today.py  # 渲染首页 / 归档 / 下载 zip
```

## 自动更新

`.github/workflows/daily.yml` 配置了每天 UTC 0:30（北京时间 8:30）自动执行生成脚本。
